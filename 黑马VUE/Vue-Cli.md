##### 全局安装 Vue-Cli

npm install -g @vue/cli

##### 创建项目

vue create my-project 

##### 配置

```
 (*) Babel
 ( ) TypeScript
 ( ) Progressive Web App (PWA) Support
 (*) Router
 (*) Vuex
 (*) CSS Pre-processors
>(*) Linter / Formatter   // 代码风格检查和格式化
 ( ) Unit Testing  // 单元测试
 ( ) E2E Testing  // E2E 测试
```
