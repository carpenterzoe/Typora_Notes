##### 分别写出以下几个HTML标签：文字加粗、下标、居中、字体

```html
<strong>文字加粗</strong>
<b>文字加粗</b>
<sub>下标</sub>   // 显示在同一行文字的右下方
<center>居中</center>   // 已废弃的标签
<font></font>
<basefont></basefont>   // 不赞成使用的标签。 定义文字的字体、尺寸和颜色。
```

##### Doctype作用？你知道多少种Doctype类型？

Doctype位于文档最前面，处于HTML标签之前。

作用：告诉浏览器，文档使用哪种HTML或者XHTML。

Doctype类型：

1. 严格版本
2. 过滤版本
3. 基于框架html版本
