##### webpack安装  两种方式

1. 安装全局webpack，这样就能在全局使用webpack的命令。

   ```javascript
   npm i webpack -g
   ```

2. 在项目根目录中运行 npm i webpack --save -dev 安装到项目依赖中。

##### dist文件夹的意义

项目做完需要发布一版，dist文件夹里的内容就是发布给用户的内容。

##### 快速生成多个 li

ul>li*10{这是第$个li}      每条li的序号即$

##### 项目初始化

```javascript
npm init -y   // -y 简单项目直接初始化
```
